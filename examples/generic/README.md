# Generic example

A small, filled-in example showing what the context files look like once the
placeholders are replaced — using neutral names only (`Project Name`,
`Module A`, `Service X`, `Feature Y`).

Use it as a reference for the level of detail and brevity to aim for. The blank
templates these are based on live in [`templates/generic/`](../../templates/generic).

Files:

- `01_PROJECT_BRIEF.md` — a filled-in project brief.
- `02_CURRENT_STATE.md` — a short, healthy current-state snapshot.
